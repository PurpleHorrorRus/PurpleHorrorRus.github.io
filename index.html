Проекты
Бесплатный и современный музыкальный плеер для социальной сети VK. Самый большой личный проект. Проект проприетарный и не преследует коммерческих целей.

Операционная система: Windows 10 и выше, Linux
Технологии: Electron, Nuxt/Vue 2, SCSS
Мессенджер для социальной сети VK с открытым исходным кодом. На данный момент проект находится в стадии бета-тестирования. Приложение предлагает возможности кастомизации и расширяет функционал при общении.

Технологии: Electron, Nuxt/Vue 2, SCSS
Приложение для ведущих прямых трансляций с одним монитором. Поддерживает полную интеграцию с площадками Twitch и Trovo. Приложение выводится поверх всех окон, имеет низкое потребление ресурсов компьютера, поддерживает вывод чата, возможность редактировать виджеты прямых трансляций, редактирование инфо...
Simple Helix API - Библиотека с открытым исходным кодом, необходимая для простой инициализации и работы с API стриминговой платформы Twitch. Имеет более 200 скачиваний на npm.

Технологии: Node.js, Jest
Паттерны: Single responsibility, Dependency inversion
